# ADIP Master Plan

## Vision

ADIP (Agentic Dealership Intelligence Platform) is a multi-tenant dealership intelligence platform designed to manage leads from intake to conversion.

## Lead Lifecycle

Lead Created
→ Validation Agent
→ Normalization Agent
→ Scoring Agent
→ Assignment Agent
→ Workflow Agent
→ Communication Agent
→ Call Intelligence Agent
→ Dynamic Re-Scoring Agent
→ Analytics

## Core Goals

* Multi-tenant architecture
* Event-driven agent communication
* Persistent audit trail
* Recoverable agent failures
* DuckDB for local development
* Supabase for production
* Dockerized deployment
* AI-assisted lead management

## Agent Responsibilities

Validation Agent

* Validate lead data
* Detect duplicates

Normalization Agent

* Standardize lead information

Scoring Agent

* Calculate lead quality score

Assignment Agent

* Assign sales executive

Workflow Agent

* Recommend next best action

Communication Agent

* Send WhatsApp communications

Call Intelligence Agent

* Analyze customer conversations

Dynamic Re-Scoring Agent

* Update lead score from interactions
